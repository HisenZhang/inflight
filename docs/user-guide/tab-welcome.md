# WELCOME Tab

The WELCOME tab provides an overview of InFlight and quick access to getting started.

## Overview

The WELCOME tab is the first screen you see when opening InFlight. It serves as:
- Landing page with app introduction
- Feature overview
- Known limitations
- Quick start guide
- Disclaimer and legal notice

## Sections

### App Title

```
INFLIGHT
FLIGHT PLANNING & NAVIGATION TOOL
```

Simple branding and tagline identifying the application purpose.

### Disclaimer

> ⚠️ **DISCLAIMER**
>
> **NOT FAA APPROVED — EDUCATIONAL USE ONLY**

**Critical Legal Notice:**

InFlight is **not certified** for actual flight operations. This means:
- ❌ Not approved by FAA or any aviation authority
- ❌ Not substitute for certified avionics
- ❌ Not for use in actual IFR or VFR operations
- ✅ Educational and training purposes only
- ✅ Flight planning practice and study

**Always use:**
- Official aeronautical charts
- Certified GPS/navigation equipment
- Current NOTAMs and TFRs
- Proper flight planning procedures
- Approved navigation tools for actual flight

### Features

Quick overview of InFlight capabilities:

**Route Planning with Airways, SID/STARs, Direct Routing**
- Plan complex routes using airways and procedures
- Support for Victor, Jet, and RNAV routes
- SID and STAR expansions

**Navigation Log with Winds Aloft, Fuel Calculations, ETAs**
- Detailed leg-by-leg flight plan
- Wind correction and heading calculations
- Fuel burn tracking and reserves
- Time estimates and ETAs

**Moving Map with Real-Time GPS Tracking**
- Vector map display
- Live GPS position
- Tactical navigation guidance
- Waypoint progression

**FAA NASR + OurAirports Database**
- ~73,000 waypoints
- Airports, navaids, fixes
- Runways and frequencies
- Airways and procedures

**Flight Tracking with GPX Export**
- Automatic flight recording
- GPS track storage
- Export to GPX format
- Compatible with aviation apps

### Limitations

Important constraints to be aware of:

**US Domestic Waypoints Only (No Oceanic/NAT Tracks)**
- Database covers US domestic airspace
- Canada, Mexico have limited coverage
- Oceanic waypoints not supported
- NAT tracks not available
- For international flight planning, use official tools

**OurAirports Data May Be Outdated**
- Community-sourced data
- Not always current
- Always verify with official charts and NOTAMs
- Use current sectionals and approach plates

**No NOTAM, TFR, or Airspace Restrictions**
- Does not include NOTAMs (Notices to Airmen)
- No TFR (Temporary Flight Restriction) data
- No special use airspace status
- Check official sources before flight

**GPS Accuracy Depends on Device Capabilities**
- Mobile GPS varies widely
- Not certified for navigation
- Environmental factors affect accuracy
- Always have backup navigation

### Quick Start

Step-by-step guide to get flying quickly:

**1. DATA — Load Aviation Database**
- Click DATA tab
- Click LOAD DATABASE
- Wait for completion

**2. ROUTE — Enter Route**
- Example: `KORD PAYGE Q822 GONZZ KLGA`
- Supports complex routing

**3. NAVLOG — Review Flight Plan**
- Check waypoints and legs
- Verify headings and distances
- Review fuel (if enabled)

**4. MAP — Enable GPS Tracking**
- Activate GPS
- Follow moving map
- Real-time guidance

**GET STARTED Button**
- Clicking jumps directly to DATA tab
- Shortcut to begin first step

## Why the Disclaimer?

**Legal Protection:**

- Clarifies intended purpose
- Meets legal requirements for flight planning tools

**User Safety:**
- Ensures users understand limitations
- Prevents reliance on uncertified tools
- Encourages proper flight planning procedures

**Regulatory Compliance:**
- FAA requires certification for approved navigation
- InFlight is not certified
- Clear labeling prevents misunderstanding
