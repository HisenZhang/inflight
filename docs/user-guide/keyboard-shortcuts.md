# Keyboard Shortcuts

Quick reference for keyboard shortcuts in InFlight.

## Navigation

| Shortcut | Action | Notes |
|----------|--------|-------|
| `1` - `7` | Switch tabs | Numbers correspond to tab order |
| `Tab` | Cycle through input fields | Standard browser behavior |
| `Shift + Tab` | Reverse cycle inputs | Standard browser behavior |

## Tab-Specific Shortcuts

### ROUTE Tab

| Shortcut | Action | Notes |
|----------|--------|-------|
| `↑` | Previous autocomplete suggestion | In route input fields |
| `↓` | Next autocomplete suggestion | In route input fields |
| `Enter` | Select autocomplete suggestion | Or submit form |
| `Esc` | Close autocomplete dropdown | Cancel selection |
| `Ctrl/Cmd + Enter` | Calculate route | When in any input field |

### NAVLOG Tab

| Shortcut | Action | Notes |
|----------|--------|-------|
| `Ctrl/Cmd + E` | Export navlog | Save as JSON |
| `Ctrl/Cmd + I` | Import navlog | Open file dialog |
| `Ctrl/Cmd + P` | Print navlog | Browser print dialog |

### MAP Tab

| Shortcut | Action | Notes |
|----------|--------|-------|
| `+` or `=` | Zoom in | Increase map detail |
| `-` or `_` | Zoom out | Decrease map detail |
| `F` | Full route view | Show entire route |
| `D` | Destination view | Zoom to destination |
| `5` | 50NM tactical view | 50NM radius |
| `2` | 25NM tactical view | 25NM radius |
| `←` | Previous waypoint | Manual navigation |
| `→` | Next waypoint | Manual navigation |

### CHKLST Tab

| Shortcut | Action | Notes |
|----------|--------|-------|
| `Space` | Toggle checkbox | When focused |
| `↑` | Previous checklist item | Navigate items |
| `↓` | Next checklist item | Navigate items |
| `Enter` | Expand/collapse section | When on section header |

### STATS Tab

| Shortcut | Action | Notes |
|----------|--------|-------|
| `Ctrl/Cmd + E` | Export track | Save GPX file |
| `Ctrl/Cmd + R` | Refresh stats | Update display |
| `S` | Start/stop recording | Manual mode only |

## Global Shortcuts

| Shortcut | Action | Notes |
|----------|--------|-------|
| `Ctrl/Cmd + R` | Reload page | Browser standard |
| `Ctrl/Cmd + +` | Increase text size | Browser zoom |
| `Ctrl/Cmd + -` | Decrease text size | Browser zoom |
| `Ctrl/Cmd + 0` | Reset text size | Browser zoom reset |
| `F11` | Fullscreen mode | Desktop browsers |
| `F12` | Developer tools | For debugging |

## Browser Standard Shortcuts

InFlight respects standard browser shortcuts:

### Text Editing

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + A` | Select all |
| `Ctrl/Cmd + C` | Copy |
| `Ctrl/Cmd + V` | Paste |
| `Ctrl/Cmd + X` | Cut |
| `Ctrl/Cmd + Z` | Undo |
| `Ctrl/Cmd + Shift + Z` | Redo |

### Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + F` | Find in page |
| `Backspace` | Go back (some browsers) |
| `Alt + ←` | Go back |
| `Alt + →` | Go forward |

## Mobile Gestures

InFlight is touch-optimized for mobile devices.

### General Gestures

| Gesture | Action | Notes |
|---------|--------|-------|
| Tap | Select/activate | Standard tap |
| Long press | Context menu | Some elements |
| Swipe left/right | Navigate tabs | Experimental |

### Map Gestures (Planned)

| Gesture | Action | Notes |
|---------|--------|-------|
| Pinch | Zoom in/out | Future feature |
| Drag | Pan map | Future feature |
| Double tap | Zoom in | Future feature |

## Customizing Shortcuts

Currently, keyboard shortcuts are not customizable. This is a potential future enhancement.

## Accessibility

InFlight supports keyboard navigation for accessibility:

- **Tab order**: Logical flow through interface
- **Focus indicators**: Visible focus states
- **Screen reader support**: Semantic HTML
- **Keyboard-only operation**: All features accessible

## Tips for Efficient Use

1. **Learn tab numbers** - Quick switching without mouse
2. **Use autocomplete arrows** - Faster route entry
3. **Keyboard-first workflow** - Minimize mouse usage
4. **Zoom shortcuts** - Quick map navigation
5. **Export shortcuts** - Save time on backups

## Shortcut Conflicts

Some shortcuts may conflict with:
- Browser extensions
- Operating system shortcuts
- Screen readers
- Password managers

If a shortcut doesn't work, check for conflicts.
