# CHKLST Tab

The CHKLST tab provides an interactive, digital pre-flight checklist system.

## Overview

The checklist feature offers a comprehensive, interactive checklist for the **Piper Cherokee 140**, designed to be used on mobile devices or tablets in the cockpit.

> ‚ö†Ô∏è **Important**: This checklist is not intended to replace manufacturer procedures. Always use your aircraft's official Pilot Operating Handbook (POH) or Aircraft Flight Manual (AFM).

## Checklist Sections

The checklist is organized into 10 phases of flight:

1. **BEFORE START** - Pre-engine start procedures
2. **ENGINE START** - Starting the engine
3. **BEFORE TAXI** - Pre-taxi checks
4. **RUN-UP** - Engine run-up and systems check
5. **BEFORE TAKEOFF** - Final checks before departure
6. **CLIMB** - After takeoff procedures
7. **CRUISE** - Cruise flight configuration
8. **DESCENT** - Preparing for landing
9. **AFTER LANDING** - Post-landing procedures
10. **SHUTDOWN** - Engine shutdown and securing

## Checklist Items Format

Each item follows this pattern:

```
Item name - EXPECTED STATE
```

**Examples:**

- `Fuel selector - DESIRED TANK`
  - Action: Select fuel tank
  - Expected: Whichever tank you want to use

- `Mixture - FULL RICH`
  - Action: Set mixture control
  - Expected: Full rich position

- `Oil pressure - CHECK`
  - Action: Verify oil pressure
  - Expected: Within green arc

## Reset Function

**RESET ALL Button** (bottom of checklist):
- Unchecks all items in all sections
- Collapses all sections except BEFORE START
- Returns to initial state
- Use between flights

**When to Reset:**
- After shutdown checklist complete
- Beginning new flight
- Practice/training sessions
- Passenger/student handoff

## Piper Cherokee 140 Specifics

This checklist is tailored to the **Piper PA-28-140 Cherokee**:

**Engine**: Lycoming O-320 (150 HP)
**Avionics**: Standard 6-pack with modern radio stack
**Configuration**: Typical trainer/rental setup

### Key Items Explained

**Before Start:**
- "Throttle OPEN 1/4"" - For priming/starting
- "Beacon ON" - Safety before prop movement

**Run-Up:**
- "Magnetos (125/50)" - Max drop 125 RPM, differential 50 RPM
- "2000 RPM" - Standard run-up power setting

**Before Takeoff:**
- "Fuel pump ON" - Electric boost for takeoff
- "Elevator trim SET" - Usually neutral or slightly nose-up

### Variations by Model

Your Cherokee may differ:
- Electric trim (some models)
- Additional fuel tanks
- Different avionics
- Modified checklist from operator

**Always use your aircraft's specific checklist from the POH.**

## Checklist Best Practices

### Pre-Flight

1. **Review complete checklist** before entering aircraft
2. **Know your aircraft** - Understand differences from generic list
3. **Have backup** - Carry paper checklist
4. **Battery check** - Ensure device charged
5. **Mounting** - Secure device before starting

### In-Flight

1. **Flow then check** - Use flow pattern, verify with checklist
2. **Read aloud** - Vocalize each item (crew coordination)
3. **Challenge-response** - PIC calls, copilot responds
4. **Do not skip** - Complete all items, every flight
5. **Do not rush** - Take time for thoroughness

### Post-Flight

1. **Complete shutdown** checklist fully
2. **Reset for next flight** when parked
3. **Secure device** - Remove from mount
4. **Charge device** for next use

## Troubleshooting

### Checkboxes Not Working

**Problem**: Can't check items

**Solutions:**
1. Try tapping item text (not just box)
2. Reload page
3. Clear browser cache
4. Try different browser

### Checklist Reset Not Working

**Problem**: RESET ALL doesn't uncheck items

**Solutions:**
1. Reload page
2. Manually uncheck items
3. Check browser console for errors

### Sections Won't Expand

**Problem**: Clicking header doesn't open section

**Solutions:**
1. Try clicking chevron icon (‚ñ∫ or ‚ñº)
