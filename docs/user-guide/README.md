# InFlight User Guide

Welcome to the **InFlight** user guide! InFlight is a comprehensive flight planning and navigation tool designed for pilots and aviation enthusiasts.

> ⚠️ **IMPORTANT DISCLAIMER**
> InFlight is **NOT FAA APPROVED** and is intended for **EDUCATIONAL USE ONLY**. Do not use this application for actual flight operations. Always use official, certified navigation tools and charts for real-world aviation.

## What is InFlight?

InFlight is a web-based flight planning application that combines route planning, navigation calculations, moving maps, and GPS tracking into a single, unified interface. It uses FAA NASR and OurAirports data to provide waypoint information for airports, navaids, and fixes across the United States.

## Key Features

- **Advanced Route Planning**: Plan routes using airways, SIDs, STARs, and direct routing
- **Wind Correction**: Automatic wind correction with winds aloft data
- **Fuel Planning**: Calculate fuel burn, reserves, and endurance
- **Navigation Log**: Detailed leg-by-leg flight plan with headings, distances, and times
- **Moving Map**: Real-time GPS tracking with tactical navigation display
- **Interactive Waypoints**: Click airport waypoints for detailed information
- **Haptic & Audio Feedback**: Device vibration and voice announcements for waypoint passage
- **Flight Tracking**: Record and export your flights as GPX files
- **Pre-flight Checklist**: Piper Cherokee 140 checklist (customizable)

## System Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for:
  - Initial database download
  - Winds aloft data
  - AirNav links
- GPS-enabled device for moving map features
- Location permissions enabled

## Limitations

Be aware of these limitations:

- **US Domestic Only**: Waypoint database covers US domestic airspace; no oceanic or NAT track support
- **Data Currency**: OurAirports data may be outdated; always verify with current charts
- **No NOTAM/TFR**: Does not include NOTAMs, TFRs, or airspace restrictions
- **GPS Accuracy**: Depends on your device's GPS capabilities
- **Educational Only**: Not certified for actual flight operations
